SECURE YOUR DESIGN WITH MICROCHIP TRUST PLATFORM

TRUST PLATFORM for CryptoAuthentication® devices

The Trust Platform for CryptoAuthentication™ is a combination of multiple elements to simplify the addition of hardware security to any embedded design.

It combines: A unique secure flow to easily protype various security use cases leveraging Microchip CryptoAuthentication™ device family (ATECC608B / ATSHA204A and future devices) and an optimized secure provisioning flow with limited Minimum Order Quantities: starting at 10 units!

This platform includes Hardware protyping tools along with a unique design suite to accelerate the prototyping of the most common use cases.

The Trust Platform is split in 3 sub-families: Trust&GO, TrustFLEX and TrustCUSTOM which allow different level of flexibility in the customization and provisioning services. Start your journey with Trust&GO and navigate to TrustFLEX / TrustCUSTOM for the highest level flexibility.


Pre-Configured and Pre-provisioned Secure Elements for fix-function Use Cases


The Microchip Trust&GO product family allows to secure a connection to any cloud infrastructure whether IP based or LoRaWAN based

  • Trust&GO for TLS (ATECC608B)

    This configuration should be used to connect to any public cloud infrastucture using a TLS based authentication. It is compatible with

    • Google Cloud Platform IoT Core
    • Amazon AWS IoT device
    • Amazon AWS GreenGrass
    • Microsoft Azure IoT Hub services

    This configuration can also be used for Accessory / Disposable asymmetric authentication

  • Trust&GO for LoRaWAN (ATECC608B)

    These configurations should be used to connect to a LoRaWAN TTI or Actility join server

    • Microchip LoRaWAN stack for TTI (SAMR34)
    • Microchip LoRaWAN stack for Actility (SAMR34)
    • ARM mbed LoRaWAN stack for TTI (SAML21)
    • LoRaMac-node stack for TTI (SAMR34)

If your application requires different or additional use cases, please consider using TrustFLEX

Select Trust&GO

Pre-configured and partially Pre-Provisioned Secure Element for Cloud Authentication


The Microchip TrustFLEX product family allows to extend the supported use cases from Trust&GO with additional Security Use Cases

  • TrustFLEX for TLS (ATECC608B)

    In addition to the Trust&GO TLS capabilities, this device supports the following use case extension:

    • Firmware validation (aka secure boot)
    • Secure Over the Air upgrade (OTA)
    • Custom Public Key Infrastructure (PKI)
    • IP / Firmware protection
    • Accessory or Disposable authentication
  • TrustFLEX for LORAWAN (ATECC608B)

    This device extends the ATECC608B LoRaWAN features (TTI or Actility) with the following use cases:

    • Customer specific DevEUI range
    • Firmware validation (aka Secure boot)

If your use cases are not supported, please consider using TrustCUSTOM

Select TrustFLEX

Secure Element to be custom configured and provisioned for Custom Usecases


The Microchip TrustCUSTOM product family allows to address the most demanding use cases. The user has full freedom in defining the device configuration and the provisioned cryptographic assets.

  • ATECC608B-TCSM

    For security schemes requiring asymmetric and symmetric cryptography

  • ATSHA204A-TCTM

    For security schemes requireing symmetric cryptography

The TrustCUSTOM Configurator tool (available under NDA) will allow you to explore configuration options and to follow a methodology to implement the most demanding embedded system security schemes

Select TrustCUSTOM

PROTYPING PLATFORM

Microchip has developed a hardware prototyping platform to directly interract with the TrustPlatform Design Suite tools for fast and easy prototyping.

CrytptoAuth Trust Platform kit- DM320118

DM320118 Product Page
DM320118 User Guide

This board comes with the 3 elements of the TrustPlatform: 1 Trust&GO, 1 TrustFLEX and 1 TrustCUSTOM device. They are all connected to an ARM Cortex-M0+ SAMD21 Microcontroller containing communication firmware to interact with the tool suite over USB.

This kit also carries an on board debugger which allows to flash the Microcontroller and live debug the application directly from MPLAB X or Atmel Studio. It also allows to plug MikroElektronika click boards.
